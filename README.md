## 🚀 Projects

### 研究

Auditory Attention Decoding without Spatial Information: A Diotic EEG Study
(空間情報なしの聴覚注意デコーディング：両耳性脳波研究)

聴きたい音を増幅するようなスマート補聴器等への応用へ向けた脳波(EEG)を用いた聴覚的注意デコーディングの研究です.
